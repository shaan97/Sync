Detailed Description

This represents a group of clients who want to synchronize music amongst themselves. In some sense, this is an encapsulation of a collection of Members

See also
shaan97::sync::Member
Note
A Group must always have at least one member. If there is no member, it is a wasted resource, and should thus be handled responsibly.
